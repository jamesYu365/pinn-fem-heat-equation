# 实验记录

## 1. 实验场景

1. 无源项齐次 Dirichlet
2. 带源项制造解问题
3. 局部热源扩散问题
4. 非齐次边界温度驱动问题

## 2. Case 1 实验配置

### 通用参数

- 热扩散率 α = 0.01
- 计算域 [0,1]²
- 终止时间 T = 0.5

### FEM 参数

- 网格大小：20×20（441 节点，800 单元）
- 时间步长：dt = 0.01（50 步）
- 边界条件：隐式 Euler 中统一施加 Dirichlet BC

### PINN 参数

- 网络结构：[3, 8, 16, 16, 8, 1]（593 参数）
- 学习率：0.005
- 训练 epoch：6000
- 配点数：域内 4000、IC 1000、BC 1000
- 损失权重：λ_r=1.0, λ_ic=100.0, λ_bc=100.0
- 验证配点：域内 2000、IC 500、BC 500
- 时间泛化：val_time_ratio=0.6（训练 t∈[0, 0.3]，外推 t∈(0.3, 0.5]）
- Early Stop：patience=1000

## 5. Case 1 PINN 反问题配置

### 反问题参数

- 初始猜测 α₀ = 0.005（真实值 α = 0.01）
- 观测数据：20 个随机传感器位置 × 每位置 10 个随机时间点，共 200 个观测点，高斯噪声 σ = 0.01
- α 参数化：softplus(log_α)，保证正性
- α 学习率：0.005（独立于网络学习率）
- 损失权重：λ_data = 10.0

### 反问题采样说明

- 观测数据使用传感器模式：先随机采样固定空间位置 `(x, y)`，再为每个传感器随机采样多个时间点 `t`。
- 当前默认配置为 `num_sensor_locations = 20`、`num_observation = 200`，因此每个传感器有 10 个时间观测点。
- 当 `val_time_ratio < 1` 时，训练和验证使用不同的传感器空间位置；训练观测位于 `t∈[0, T_train]`，验证观测位于外推时间段 `t∈[T_train, T_end]`。
- PDE 域内训练配点为均匀随机采样；IC 初始条件点为分层 jittered grid 采样；BC 边界点为四条边上的分层随机采样。
- 验证配点在训练开始前独立采样。一次实验运行中，训练点和验证点均固定，不会在每个 epoch 动态重采样。

### α 偏差排查

若反问题最终 α 估算值明显偏离真实值，优先按以下方向排查：

- α 是否真的在更新：记录 `log_alpha.grad` 或 `alpha_grad`，确认 α 梯度不是长期接近 0。
- 观测时间是否有足够识别信号：如果观测集中在早期时间，温度衰减差异较弱，模型可能靠网络本身拟合数据而不需要调准 α。
- 观测设计是否稳定：下一阶段可优先尝试“固定传感器位置 + 每个传感器使用均匀时间网格”，替代完全随机时间点。
- 损失权重是否平衡：小范围扫描 `lambda_data = 10, 50, 100`，观察 data loss、PDE loss 与 α 误差的变化。
- α 学习率是否过小：小范围扫描 `alpha_lr = 0.005, 0.01, 0.02`，尤其关注 α 是否仍停在初始值附近。
- 网络是否过度吸收误差：若网络容量过强或 PDE 约束偏弱，可能拟合观测但参数不可辨；可尝试增加 collocation 点或采用先固定 α 预训练、再解冻 α 的两阶段训练。
